Website Voucher Suite

How to use

After installing the modul, you'il find the menu 'voucher' in the sales-module.

There you have a list of all voucher you addet. You can also use import.

After creating a new voucher or clicking on an existing one, the following dialog appears:

  • Name: Displayed to the customer once they have added the voucher in the online store.
  • Voucher Product: A product that represents the voucher. The price is overwritten with that of the voucher.
  • Coupon-Code: By entering this code, the customer can add the voucher to the shopping cart. This is generated when the voucher is created, but can be edited.
  • Coupon Amount: The value of the voucher. Under 'Amount Type' you can select whether this value is a fixed amount or a percentage.
  • Minimal Amount: The voucher becomes valid only when the customer has reached this amount.
  • Maximum Amount: Only if percent is selected. This is the maximum amount of the voucher.
  • Product Valid: Only valid for selected products. If there is no such product in the shopping cart, these products will be suggested.
  • Customer: The voucher is only addable for these customers
  • Start Date/Expiry Date: During this time the voucher will be applicable
  • Apply limitation: Maximum availability of the voucher. This value decreases after each picking. -1 = Not available anymore.
  • Coupon History: All orders with this voucher are displayed here. Clicking on 'Release Voucher' will remove this voucher from all open orders.

Frontend

The customer can enter the coupon code in the order completion. Depending on the availability of the voucher there are 3 scenarios:

  • Will be rejected: Not active, code wrong, wrong customer, out of time-range.
  • Accepted but not discounted: Amount not reached, Wrong discount product (Valid products are displayed).
  • Accepted: Right Voucher value discounted, Amount higher than allowed (Max Amount discounted).

The customer can remove the voucher any time. If the customer leaves the shopping cart without removing the voucher, all vouchers can be released in the backend 'Release Voucher'.
